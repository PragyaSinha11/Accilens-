# AcciLens 🔍 — Road Accident Intelligence Platform

AcciLens is a final-year college capstone project that analyzes road accident
data from Indian government sources (MoRTH) to uncover patterns in when, where,
and why accidents occur across India.


## 📌 What It Does
- Embeds an interactive Power BI dashboard with real accident data
- Visualizes accident trends by hour, month, day, road type, and weather
- Highlights the top accident-prone states with severity indicators
- Identifies peak risk windows and high-frequency accident zones

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Chart.js (data visualizations)
- Power BI Embedded (interactive dashboard)
- GitHub Pages (hosting)

## 📂 Project Structure

## 📊 Dataset
- Source: Ministry of Road Transport and Highways (MoRTH), India
- Records: 50,000+ road accident entries
- Coverage: 28 States and Union Territories
